A brief overview of the technologies behind the ORB platform.

All the code is released under an GNU GPL v3 open source license and can be found on GitHub.

  • the core of the code is written in the Django framework
  • the front-end interface uses the Bootstrap 3 framework
  • the REST API has been developed using the Tastypie framework
  • the search functionality is powered by Apache Solr

Information on how to use the ORB platform API can be found on ReadTheDocs
